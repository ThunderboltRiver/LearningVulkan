# Learning Vulkan And RealTimeRendering For GameDevelopment

## 概要

Vulkanやゲーム開発のためのリアルタイムレンダリング周辺技術について学習するための個人的なリポジトリです。
バイオハザードやCODなどの著名タイトルのようなフォトリアルなリアルタイムレンダリングを行うための技術について、マジの完全な初心者（自分）が必死こいて勉強しているという記録を日報やコードに残すだけです。
決して学習教材が揃っているリポジトではないので注意。

予定としては主に以下のことを（私が）学んでいく予定です

- リアルタイムレンダリングの技法やトピック全般
  - PBR、シェーディング、ライティング、AO、AA, Clustered Forward/Differed, などなど

- グラフィックスAPIとシェーダ言語
  - Vulkan & Slang

- GPUの挙動についての話題
- 数学
- 物理

その日のうちに学んだことはDailyReportsなどの日報に書くことにします。
労働時間の制約上、どうしても平日は無理かもしれないが毎日何かしらコミットできたらいいな・・・という希望を込めます。
